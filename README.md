## Run Locally

Clone the project

```bash
  git clone https://github.com/kevalCoder9106/Password-Generator
```

Go to the project directory

```bash
  cd Password-Generator
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run dev
```
